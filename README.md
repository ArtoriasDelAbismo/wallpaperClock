# Wallpaper Clock

Custom Wallpaper that updates every minute. Developed with NodeJs

![App Screenshot](https://github.com/ArtoriasDelAbismo/wallpaperClock/blob/master/wallpaper.png?raw=true)

## Installation

```bash
  npm install
  pm2 start index.mjs
```
